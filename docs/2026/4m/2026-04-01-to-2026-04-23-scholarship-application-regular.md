# 2026年4月1日(水) 〜 4月23日(木) 実施 — 春の定期採用 奨学金出願受付

| 項目 | 内容 |
|------|------|
| イベント名 | 2026年度「日本学生支援機構 給付奨学金・立命館大学学費減免」および「日本学生支援機構 貸与奨学金」定期採用出願 |
| 対象 | 学部生（新入生含む。ただし一部対象外あり） |
| 出願期間 | 2026年4月1日(水) 9:00 〜 2026年4月23日(木) 17:30 |
| 出願方法 | 学生オフィス窓口で募集要項一式を入手し、出願 |
| 注意事項 | 期限厳守、出願手順・申請内容の誤りに注意、必要書類の準備、同種奨学金への重複出願不可等 |
| 詳細 | [奨学金案内ページ](https://www.ritsumei.ac.jp/scholarship/news_economic/article.html/?news_id=452) |
| 問い合わせ先 | 立命館大学 学生オフィス奨学金係 TEL: 075-465-8494（平日9:30～11:30、12:30～17:00、火曜のみ12:30～17:00） |

## 概要

2026年度「日本学生支援機構 給付奨学金・立命館大学学費減免」および「日本学生支援機構 貸与奨学金」の定期採用出願受付が開始されました。出願希望者は学生オフィス窓口で募集要項を入手し、期限内に出願してください。新入生で高校予約採用候補者はWebサイトで手続き方法を確認してください。

### 注意事項
- 出願は期限厳守です。
- 既に奨学生資格を持つ場合、同種奨学金への出願不可。
- 必要書類の準備を早めに行ってください。
- 給付奨学金の基準を満たさず学費減免のみ希望の場合も給付奨学金申込が必要です。
- 給付奨学金と貸与奨学金第一種を併給する場合、貸与月額が調整される場合があります。
