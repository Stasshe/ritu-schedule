# 2026年3月30日 公開 — 【英語団体受験】2026年度スケジュール公開のお知らせ

学生のみなさんが自分自身の英語力をはかるため、以下の団体受験（大学主催）を実施しています。

- TOEIC(R) L&R IP テスト（オンライン／マークシート）
- TOEFL ITP(R) テスト（デジタル版）

2026年度の実施スケジュールをホームページで公開しました。留学や大学院進学、就職活動などのキャリア準備にぜひ活用してください。学部により受験の義務化や受験料補助の制度があります。

◇ オンライン（TOEIC／TOEFL）の団体受験（詳細）

https://www.ritsumei.ac.jp/gengo/gaikokugo-gakusyu/dantai.html

◇ マークシート（TOEIC）の団体受験（詳細）

https://www.ritsumei.ac.jp/file.jsp?id=614918

立命館大学 言語教育センター（OIC窓口）
